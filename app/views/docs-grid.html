Grid

Overview

Tink includes a responsive, mobile first fluid grid system.

12 columns are available, degrading to six or less on small screen devices like tablets and smartphones.

Pre-defined classes are available for easy layouting, prototyping and design in the browser, plus mixins and extends for generating more semantic container names.

How to apply the grid?

Create a page layout with the provided grid and the rows and columns at your disposal. Here's how it works:

  • Always start off with a .container or .container-fluid class
  • Use rows to create horizontal groups of columns
  • Content should be placed within columns, and only columns may be immediate children of rows
  • Pre-defined classes like .row and .col-xs-4 are available. Find a full list below.
  • These grid classes apply to devices with screen widths greater than or equal to the breakpoint sizes, and override grid classes targeted at smaller devices. So a .col-md-class will not only affect it's styling on medium devices but also on larger devices if a .col-lg-element is not present.

Media Queries

Media Queries make it possible to target elements within a specific size of viewport. E.g. the title should appear red on a small screen whose width is 320px, green when the screen is 600px or more and blue whenever the screensize is bigger than 1000px.

So, media queries, make it possible to apply different styles on elements, based on certain conditions.

@media (min-width: @screen-md-min) { ... }

Column Wrapping

If more than 12 columns are placed in a row, each group of extra columns will wrap onto a new line.

.col-xs-9
.col-xs-4
Since 9 + 4 = 13 > 12, this 4-column-wide div gets wrapped onto a new line as one contiguous unit.
.col-xs-6
Subsequent columns continue along the new line.

Responsive Column Resets

With the four tiers of grids available you're bound to run into issues where, at certain breakpoints, your columns don't quite clear right, or one is taller than the other. Use a combination of .clearfix and responsive utility classes to fix this.

Nesting Columns

To nest your content, simply use the .row-class again and a set of .col-sm-*-classes. Nested rows should include a set of columns that add up to 12.

Level 1: .col-sm-9
Level 2: .col-xs-8 .col-sm-6
Level 2: .col-xs-4 .col-sm-6

Column Ordering

to change the order of columns easily, use the built-in helper classes like .col-md-push-* and .col-md-pull-*

.col-md-9 .col-md-push-3
.col-md-3 .col-md-pull-9
